# Изменяемые параметры
## Режим BlackBox SSD

*Эмуляция коммерческих SSD с* ***device-managed FTL***
```bash
./run-blackbox.sh
```
***Ключевые параметры:***
```
secsz=512              # размер сектора в байтах
secs_per_pg=8          # количество секторов на страницу
pgs_per_blk=256        # количество страниц на блок
blks_per_pl=256        # количество блоков на плоскости (plane)
luns_per_ch=8          # количество логических элементов (LUNs) на канал
nchs=8                 # количество каналов

pg_rd_lat=40000        # задержка чтения страницы (нс)
pg_wr_lat=200000       # задержка записи страницы (нс)
blk_er_lat=2000000     # задержка очистки блока (нс)

gc_thres_pcent=75      # отсечка вызова сборщика мусора (%)
```

## Режим WhiteBox SSD (OCSSD)

*Эмуляция OpenChannel SSD с* ***host-managed FTL***
```bash
./run-whitebox.sh
```
***Ключевые параметры:***
```
OCVER=2                   # версия OCSSD

ssd_size=4096             # размер диска в мегабайтах
num_channels=2            # количество каналов
num_chips_per_channel=4   # количество чипов NAND на канал
```
